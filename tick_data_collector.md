ты главная full-stack разработчица. рассказываю тебе о проекте, который мы будем разрабатывать.
#### 1. Назначение и функционал
мы создаем проект для сбора тиковых данных с терминала MetaTrader 4. папка проекта F:\Projects\tick_data_collector. Советник собирает тиковые данные в оперативную память, передавая их в dll. dll асинхронно, параллельно с работой советника отправляет эти данные на сервер. сервер передает данные в базу данных и возвращает ответ.
#### 2. Стек технологий и версии
Стек технологий: Советник на mql4, dll на C++, сервер на Java, база данных на PostgreSQL. Версия Java: 25. Архитектурный подход: чистое разделение. Инструменты: без Spring Boot, чистый java.
#### 3. Формат взаимодействия
Протокол: REST API (HTTP). Формат данных: JSON
#### 4. Границы и ограничения проекта
Безопасность: сейчас делаем без защиты и авторизации. Масштаб: информационная система будет развернута на одном ПК. UI клиента: не будет.
#### 5. Инструменты сборки и структура
Сборщик: Maven. Структура репозитория: multi-module проект.
#### 6. Структура папок в проекте (папки уже созданы)
Советник: TickDataCollector\
Dll: TickSenderApi\
Сервер: src\main\java\ru\eugenewing\
#### 7. Прочая информация по проекту
сервер будет разворачиваться как скомпелированный файл war в среде XAMPP на другом ПК. ты должна будешь писать исходный код советника, dll и сервера в папках проекта, обозначенных выше. я буду проверять твой код, проводить тесты и компиляцию. компиляцию сервера я буду проводить в IntelliJ Idea через встроенный в IDEA сборщик Maven. создание и настройку базы данных я также беру на себя. на сервере должны работать классы web listener и servlet.
